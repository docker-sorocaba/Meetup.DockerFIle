Agenda

Introdução ao Docker

9h - 9h40

Marcos Vinicius Rezende (AMENERGIA)

A palestra tem como objetivo de explicar e mostrar na pratica o conceito básico de funcionamento e instalação do docker.

  • Instalar Docker
  • Comandos Basicos
  • Subindo a Primeira Imagem
  • Portainer, Ambiente gráfico pra Docker

Integrando Docker com Jenkins

9h40 - 10h20

Eduardo Fonseca (CI&T)

A palestra irá demonstrar como realizar as configurações necessárias para se trabalhar com o Docker utilizando a ferramenta de CI Jenkins. Além disso, demonstrará como criar um Jenkinsfile para execução de containers Docker como slaves

Extra: demonstrar a utilização da ferramenta Cadvisor (Google) para monitoramento de containers em tempo real

Coffee Break

10h20 - 10h40

Porque ninguém é de ferro

Aplicação na Nuvem com GKE

10h40 - 11h20

Rodolfo Neto (GFT)

  • Breve fala sobre o que é o Kubernetes;
  • Breve fala sobre o que é o Google Kubernetes Engine (GKE);
  • Escolhendo uma aplicação web para migrá-la para a nuvem;
  • Configurando todo o ambiente do GKE via portal (painel de controle) do Google Cloud;
  • Integrando a aplicação com o GKE;
  • Automatizando as imagens docker da aplicação e construindo automaticamente.

Utilizando Infraestrutura Ágil com Práticas DevOps

11h20 - 12h

Gabriel Policante Araújo (4Linux)

Se o desenvolvimento de software é ágil a infraestrutura também deve ser. Já imaginou uma infraestrutura sendo escalável, automatizada, padronizada e versionada da mesma forma que acontece com o desenvolvimento de softwares?
